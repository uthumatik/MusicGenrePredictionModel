## Usage

Create a virtual environment:

```bash
python3 -m venv .venv
```

and activate it:

```bash
source .venv/bin/activate
```

or:

```bash
source .venv/Scripts/activate
```

Install the dependencies when the .venv is activated:

```bash
pip install -r requirements.txt
```

and to deactivate it when you're done:

```bash
deactivate
```
